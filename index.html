    • PARA QUE SERVE

    O Software Parceirão On foi desenvolvido com foco no geranciamento de dados em setores administrativos; Atualmente ele está dedicado a setores ligados a logistica, tendo as seguintes funções:

    • Cadastro e gerenciamento de motoristas
    • Cadastro e gerenciamento de rotas
    • Gerenciamento de cargas
    • Simulador de rotas
    • Gerenciamento de transferências
    • Gerenciamento de clientes
    • Gerenciamento de reentregas
    • Gerador de relatórios

    COMO FUNCIONA

    O Parceirão On usa pastas compartilhadas para armazenar os dados de forma compartilhada, tornando os dados acessiveis atravez de diverssos computadores de forma simultanea.

    COMO INSTALAR

    Siga os passos abaixo para a instalaçao do programa:

    1. Faça o dowload do programa.
    2. Execute o instalador.
    3. Digite a senha de acesso.
    4. Siga clicando em avançar.
    5. Marque a opção de criar atalho na area de trabalho.
    6. Após a instalação vá até a area de trabalho.
    7. Clique com o botão direito do mouse.
    8. Clique em ir para o local do arquivo.
    9. Abra o arquivo ways.txt.
    10. Troque os caminhos pelos caminhos das pastas de bancos de dados.
